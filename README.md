Problem statement :
Given a number of days n (1 ≤ n ≤ 10), print all possible sequences of daily activities using the set:
{"Swimming", "Running", "Football"}
subject to the constraint that no two consecutive days have the same activity.


Example :
input :
2

output :
Swimming Running
Swimming Football
Running Swimming
Running Football
Football Swimming
Football Running
COUNT: 6
